# Cálculo Interativo · Prof. Davi Rocha

Visualizadores interativos para o ensino intuitivo de Cálculo.

## Apps disponíveis

- **Explorador de Limites** — Secante → Tangente, o conceito de Limite em ação

## Como rodar localmente

```bash
npm install
npm run dev
```

## Como publicar

O deploy é automático via GitHub Actions a cada push na branch `main`.

O site fica disponível em: `https://davimatfacil.github.io/calculo-interativo/`
